End of Semester 2

Mar 31, 2021 5 min read

It’s been a while since I posted anything - semester 2 got away from me a little bit. I thought I would post to sum up how the semester 2 classes went. And reflect a bit on what I am currently working on.

Semester 2

I was involved in teaching two classes this semester: Mathematical and Statistical Computing and Spatial Statistics. Both of these have had feedback collected, so I’m using this to reflect on how the semester went and what changes can and should be made to reflect the comments.

Mathematical and Statistical Computing

Mathematical and Statistical computing is an introductory programming class (using R) delivered to second year Maths and stats undergraduate students. This course is usually particularly challenging because it is different to any other classes that they have taken before. I made the decision to be upfront about this. In the introductory session of the class I said that they should expect to find the class hard, and they might feel as though they are struggling, but to keep pushing and asking questions.

My colleague and I also decided to spread the materials over more weeks than was given for the MSc students in semester 1, so that they had more time to process the typical threshold concepts (loops - everybody hates loops!) This was definitely wise and the students seemed to appreciate this.

The feedback for the class was overall very positive. They felt able to ask questions where they were struggling (which was the goal) and gradually got more independent as the weeks went on. This was really nice to see - I actively encouraged using google and stack exchange as a resource since I knew it would be impossible for me to personally teach every single aspect of R for every possible application. I also wanted them to develop the search engine skills to solve problems on their own. This started out as a grumble, but the students got quite comfortable with it quite quickly.

The key criticism of the course was the desire for more video content. I suspect that this was from students who did not attend the live session (although I have no way of knowing that for sure). In the live session I took the key difficult points that were identified in the tutorial session or through student feedback and discussed problems that were relevant to them. These sessions were recorded and placed on the class page, but did not get a huge number of views.

Spatial Statistics (Face-to-Face)

Spatial statistics is designed to give MSc students an insight into analysing spatial data. I am going to be putting this online for a cohort starting in May, so I was filming for that in time to get the videos online for the face to face version. I didn’t want to duplicate effort, but that was definitely an intense few weeks!

Students usually find this course quite challenging, so I designed it in a fairly structured way that would help to scaffold towards them being able to do a full data analysis. The theoretical concepts were always delivered first via video. Then the methods for applying the theory (in R) were delivered via sets of notes which additionally contained short programming exercises and quizzes. Following this there would be an interactive video where I would demonstrate the code on a data set and ask them to think about their interpretation of the data (which I would then discuss in the video). Then each week they would peform an analysis on spatial data within a supported tutorial session (via Zoom). This was then discussed in more detail in a live session the following week.

The class got probably the best feedback I’ve ever had. Students appreciated the content that was given and found the explanations really useful. They commented specifically on my use of the VLE being varied and interesting.

The key aspects for development based on this were to provide additional description of the context in which you would use the analysis techniques. I did this at the time via forum posts in the weekly Q&A forums, and will build this into future iterations of the class.

Current Work

I am going to be delivering both of these classes in the near future for different audiences. I’m delivering an introductory R class as CPD starting in April, and delivering Spatial Statistics online starting in mid-May to a small cohort of online MSc students.

Introduction to Programming in R (CPD)

This will run with roughly the same materialas before. It is anticipated that this will run entirely online, so I will be thinking about how to present the live sessions that I would have done face-to-face.

I will try to run a weekly drop in session, but may also make the tutorial sheet something that is submitted, so that I can provide some discussion around key mistakes.

Spatial Statistics (Online)

The materials for this will be almost identical to those developed for the face-to-face version of the class. The key difference is that the tutorial material for the face-to-face students will become a student submission which I will provide feedback on. For small class sizes, this will be individual, but as class sizes grow, this will be more general feedback to the entire class.

I may re-evaluate this in the future if I can find a satisfactory way of giving detailed feedback to each student.
