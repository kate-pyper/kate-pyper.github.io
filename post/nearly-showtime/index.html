It's Nearly Showtime

Sep 14, 2020 4 min read

We’ve just entered week 0 (the induction week) of the semester here at Strathclyde, and I’ve been working more than usual trying to get good quality content online for the upcoming semester. The first two weeks’ worth of material is ready to go but there’s still 8 weeks left to do - I’m definitely not kidding myself that this semester is going to be straightforward.

This is going to be a reasonably short post, but i’m going to pin point some things that I’ve been doing over the past few weeks (some of which to help the students and some because I’m a perfectionist and can’t help myself).

Making materials for offline as well as online

I’ve spent quite a lot of time making text resources because these transfer a bit easier to students with limited internet access, and some things are just easier to explain in text than in a video. I’ve used R-markdown to create this text content, because it allows me to create both html and pdf documents. I’ve structured my course so that each small topic comes with text online, generally with some exercises, and then has an associated downloadable pdf document that contains all of the information from the online activity.

Since the online text contains embedded exercises. I want to encourage the students to work through these. I’ve done this firstly by setting activity completion on the online materials but not on the downloadable materials; and secondly by restricting access to the downloadable materials until the online activity has been attempted (though if a student were unable to access the online activity I would provide access to the PDF document). Consistent but not consistent

I’ve tried to keep a fairly consistent weekly structure, with each week being visually similar to each other. Each week starts with a description of the learning objectives, followed by the materials and links to the live zoom sessions that are available each week (see below).

The end of the week is also pretty consistent, and consists of the student reflection exercise discussed in one of my previous posts, as well as a weekly formative assessment in the form of either a quiz or a submission as shown below.

During the week the students will work through a number of activities (I’m aiming for somewhere between 6 and 10). Those which contain a lot of information are contained in lessons and those which contain smaller (but no less important) points are contained within pages as demonstrated below.

Within these activities I have tried to give a bit of variety. There are large elements of text and examples, which are interspersed with the occasional video where I think these are useful. These will also contain some sort of associated activity, be that an online programming activity, or an interactive H5P activity.

Overall the goal is to be consistent, though not so consistent that the students get bored.

Formatting

Something that I’ve found myself enjoying quite a bit recently (and something that I’ve become a bit obsessive about) is formatting my materials nicely in MyPlace/Moodle. As I mentioned, I have been using R-markdown to produce HTML materials. I then use the output from that and copy it into the html editor within MyPlace - this is a bit hidden you need to click on the button for more options and then click on the button with </> shown on it.

One thing to not about this is that MyPlace has been created so that it is difficult to tamper with the themes, which means that it is hit-or-miss whether CSS will work. So I’ve had to do the formatting old school and use the style attribute of various HTML tags. I’ve saved the commonly used styles into a text document to make this a bit easier. Video Editing

The final thing that I have been doing over the past few weeks is editing video. I’ve used zoom to record since it’s easy to get a video of both your face and a shared screen (be that slides or R). You can download a video that contains the shared screen alongside you speaking in a shared box, but I wasn’t particularly keen on that. My perfectionist tendencies kicked in here and I decided to try my hand at video editing, so that I could edit out my mistakes and edit together the video stream with my face in it and the video stream showing the shared screen. I’ve been using shotcut which is free and also relatively straightforward once you get the hang of it!
